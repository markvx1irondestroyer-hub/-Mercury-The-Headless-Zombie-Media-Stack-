## Installation
1. Download the files to your home directory.
2. Run the installer:
   `bash install.sh`
3. After reboot, enter your mercury folder:
   `m cd`
4. Fire up the stack:
   `m up`

## The 'm' Command
| Command | Result |
|---|---|
| `m up` | Starts all media apps |
| `m gpu` | Monitors GPU Temp/Usage |
| `m ps` | Shows what is running |
| `m reset` | Nuclear wipe and restart |
